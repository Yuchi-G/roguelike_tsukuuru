# Roguelike Tsukuuru

TypeScript + Vite + HTML Canvasで作る、ローグライク制作に特化した小型ミニゲームエンジンです。汎用エンジンではなく、グリッド移動、ターン制、戦闘、ダンジョン生成など、ローグライクに必要な最小機能をシンプルにまとめています。

## 現在できること

- ランダムな部屋と通路で構成されたダンジョンを生成
- `@`のプレイヤーを矢印キーまたはWASDで移動
- 壁`#`には移動不可
- 敵にぶつかると攻撃
- プレイヤー行動後に敵が1回動く完全ターン制
- HP、攻撃力、死亡、ゲームオーバーを管理
- 回復薬`!`を拾ってバッグに保管
- バッグ内の回復薬を使ってHP回復
- バッグの上限は10個。満杯時は拾うアイテムと捨てるアイテムを選択
- 武器`)`を拾うと自動装備して攻撃力を強化
- 階段`>`から次の階へ移動
- 次の階へ進む時にHP、レベル、経験値、攻撃力、武器を持ち越し
- 敵を倒すと経験値を獲得し、一定値でレベルアップ
- レベルアップ時に最大HPが増え、HPが全回復し、攻撃力が上昇
- 複数種類の敵がランダムに出現
- 5階ごとに敵が少し強化
- プレイヤー周囲8マスの簡易視界
- 画面右側にLV、EXP、HP、攻撃力、武器、バッグ、階層、ログを表示

## セットアップ

```bash
npm install
```

## 開発起動

```bash
npm run dev
```

Viteが表示するローカルURLをブラウザで開くとゲームをプレイできます。

## ビルド

```bash
npm run build
```

## 操作

| 操作 | 内容 |
| --- | --- |
| 矢印キー / WASD | 移動 |
| 敵に向かって移動 | 攻撃 |
| 回復薬の上に移動 | 回復薬をバッグに入れる |
| 武器の上に移動 | 武器を拾って自動装備 |
| H / UIの「使う」ボタン | バッグ内の回復薬を使う |
| UIの「中身を見る」ボタン | バッグの中身を表示 |
| 階段の上でSpace | 次の階へ進む |
| ゲームオーバー時にEnter | 新しいゲームを開始 |

## 画面記号

| 記号 | 意味 |
| --- | --- |
| `@` | プレイヤー |
| `s` | スライム |
| `g` | ゴブリン |
| `O` | オーク |
| `!` | 回復薬 |
| `)` | 武器 |
| `#` | 壁 |
| `.` | 床 |
| `>` | 階段 |

## 成長システム

プレイヤーは敵を倒すと経験値を獲得します。初期状態はLv.1、EXP 0、次のレベルまで10 EXPです。

レベルアップ時は以下のように成長します。

| 項目 | 変化 |
| --- | --- |
| LV | +1 |
| EXP | 必要経験値分を消費 |
| 次の必要EXP | 現在値の1.5倍、切り捨て |
| 最大HP | +5 |
| 現在HP | 最大HPまで回復 |
| 攻撃力 | +2 |

## 敵の種類

敵は出現時に以下からランダムで選ばれます。5階ごとにHPと攻撃力が追加で強化されます。

| 種類 | 記号 | HP | 攻撃力 | EXP |
| --- | --- | ---: | ---: | ---: |
| スライム | `s` | 8 | 2 | 4 |
| ゴブリン | `g` | 10 | 3 | 5 |
| オーク | `O` | 16 | 5 | 9 |

## 装備システム

プレイヤーは武器スロットを1つ持ちます。武器を拾うと自動で装備され、基礎攻撃力に武器の攻撃力が加算されます。

| 武器 | 記号 | 攻撃力 |
| --- | --- | ---: |
| 剣 | `)` | +3 |

## アイテムバッグ

プレイヤーは回復薬をバッグに入れて持ち歩けます。上限は10個です。`H`キーまたはUIの「使う」ボタンで、バッグ内の回復薬を1つ使ってHPを回復します。UIの「中身を見る」ボタンでバッグ内のアイテム一覧を表示できます。

バッグが満杯の時に回復薬を拾うと、マップ画面上に半透明の選択UIが表示され、以下を選べます。

| 選択 | 内容 |
| --- | --- |
| これを捨てる | 選んだバッグ内アイテムを捨て、拾った回復薬をバッグに入れる |
| 拾わず捨てる | 拾った回復薬をバッグに入れず捨てる |

## ディレクトリ構成

```text
src/
  engine/
    Game.ts              ゲーム全体の状態管理
    Renderer.ts          Canvas描画
    InputManager.ts      キーボード入力
    TurnManager.ts       ターン管理
    Map.ts               2Dグリッドマップ
    Tile.ts              タイル定義
    Entity.ts            Entity基底クラスとActor
    Collision.ts         衝突判定
    DungeonGenerator.ts  ランダムダンジョン生成
    Fov.ts               簡易視界
    Logger.ts            ゲームログ
  game/
    Player.ts            プレイヤー
    Enemy.ts             敵データと単純追跡AI
    Item.ts              回復薬
    MainScene.ts         ゲーム初期化と階層管理
  main.ts                エントリーポイント
  style.css              UIスタイル
```

## 設計メモ

- 外部ゲームライブラリは使わず、Canvas APIで描画しています。
- エンジン部分は`src/engine`、サンプルゲーム固有の実装は`src/game`に分けています。
- 敵AIは最短経路探索を使わず、プレイヤーとのx/y差分で単純に近づきます。
- 初期MVPのため、セーブ機能、オンライン機能、3D描画は実装していません。
