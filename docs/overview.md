# 全体概要

このプロジェクトは、TypeScript + Vite + Electron + HTML Canvas で作られた、デスクトップ専用のローグライク制作アプリです。

起動直後はゲームを開始せず、設定画面でプレイヤー、敵、アイテム、階層ルール、描画、ログ文言などを編集します。ゲーム開始後は、ランダム生成されたダンジョンで、プレイヤーが1回行動し、その後に敵が順番に行動するターン制で進みます。

# 処理の流れ

1. Electron がウィンドウを作り、`index.html` を表示する。
2. `src/main.ts` が DOM 要素を取得し、`Game`、`MainScene`、`ConfigPanel` を作る。
3. 設定画面で「ゲームスタート」を押す。
4. `MainScene.loadDungeonFloor()` がダンジョン、プレイヤー、敵、アイテムを生成する。
5. `Game.startDungeonFloor()` が階層を開始し、入力を有効化して描画する。
6. プレイヤーが移動、攻撃、アイテム取得を行う。
7. プレイヤー行動後、`runEnemyTurn()` が敵を順番に動かす。
8. `Game.renderGameState()` が視界、Canvas、ステータス、ログを更新する。
9. 階段の上で Space を押すと次の階へ進む。
10. プレイヤーの HP が 0 になるとゲームオーバーになる。
